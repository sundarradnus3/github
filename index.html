Description:
This project is a 3D donut swipe prototyping animation crafted in Figma, designed to illustrate innovative UI/UX design capabilities. The project centers around a 3D donut that users can swipe, bringing to life smooth animations and interactive elements within a web or mobile interface.
The design leverages Figma’s powerful prototyping tools to create a dynamic and engaging user experience. This prototype is part of my design portfolio, showcasing my skills in creating interactive and visually striking interfaces. It demonstrates how motion and 3D elements can enhance user engagement and improve the overall usability of a digital product.

Swipe Animation:
Users can swipe the donut, triggering smooth and responsive animations that illustrate the seamless transition between different UI states.
Prototyping Excellence:
The project utilizes Figma’s prototyping capabilities to create a fully interactive experience, highlighting the potential of animated UI elements.
Responsive Design:
The interface is optimized for various devices and screen sizes, ensuring a consistent experience across platforms.
User Engagement:
The design prioritizes user interaction, making the prototype both functional and enjoyable, enhancing the overall user experience.
